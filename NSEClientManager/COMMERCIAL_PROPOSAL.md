## Platform Overview

### Core Features
- **Real-Time Results Calendar**: Date-wise earnings announcements with live status updates
- **Automated NSE Data Scraping**: Continuous polling and PDF parsing of quarterly results
- **3-Window Analysis Interface**: 
  - Quarterly Performance (QoQ/YoY comparisons)
  - Interactive Candlestick Charts (21+ days with EMA, RSA, Volume indicators)
  - Delivery vs Trading Volume Analysis
- **Multi-User Management**: Admin panel with subscription controls (Active/Demo/Inactive)
- **Advanced Filtering**: By sector, date, status, price change
- **Subscription Management**: Demo periods, activation controls

### Technology Stack
- **Frontend**: React 18 + TypeScript, shadcn/ui components, TanStack Query
- **Backend**: Node.js + Express, PostgreSQL (Supabase), Drizzle ORM
- **Scraping**: Axios, Cheerio, PDF parsing libraries
- **Real-time**: WebSocket for live status updates
- **Hosting**: Cloud-ready (Railway, AWS, DigitalOcean)
